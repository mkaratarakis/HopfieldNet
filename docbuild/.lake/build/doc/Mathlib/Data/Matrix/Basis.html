    Matrices with a single non-zero element. #

    This file provides Matrix.single. The matrix Matrix.single i j c has c at position (i, j), and zeroes elsewhere.

    source
    def Matrix.single {m : Type u_2} {n : Type u_3} {α : Type u_6} [DecidableEq m] [DecidableEq n] [Zero α] (i : m) (j : n) (a : α) :
    Matrix m n α

    single i j a is the matrix with a in the i-th row, j-th column, and zeroes elsewhere.

    Equations
    • Matrix.single i j a = Matrix.of fun (i' : m) (j' : n) => if i = i' ∧ j = j' then a else 0
